**physLP**

----

**A library providing tools for basic physics.**

   Author        : Lancelot PINCET

**physLP** is a library written by **Lancelot PINCET**.
The source code can be consulted on its `GitHub repository <https://github.com/LancelotPincet/physLP>`_.
However for simply using the features, you can install it in your Python environment using `pip <https://pypi.org/project/physLP>`_.
And for information, do not hesitate to consult the `documentation <https://physLP.readthedocs.io>`_.

1. New to **physLP**? Check out the :doc:`Beginner Guide <beginner_guide>` to get up and running.
2. Want a deeper understanding of key concepts in **physLP**? Dive into the :doc:`User Guide <user_guide>` to get an in-depth understanding of the tool.
3. Have a doubt on a syntax in **physLP**? Consult the :doc:`Reference Guide <reference_guide>` to have detailed information on how to call the objects.
